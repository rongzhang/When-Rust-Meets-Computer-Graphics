工具安装

Linux用户

  • 下载Vulkan SDK ⭐点击这里
  • 下载Rust 🚀点击这里
  • 下载一个IDE 推荐Clion或VSCode
  • 如果使用了VSCode 请安装 Rust (rls) 和 Better TOML

Windows用户? 想用DX12?

  • 安装visual studio 2019 🔥点击这里
  • 点选所有跟C++有关的东西,打开游戏开发那里安装DX调试器
  • 然后安装Rust 🚀点击这里
  • 用Vulkan同Linux
